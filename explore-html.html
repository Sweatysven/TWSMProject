Hypertext Markup Language (HTML)

HTML Introduction

A commonly used markup language, for building web sites and applications, is the Hypertext Markup Language (HTML). HTML is however not the only language webmasters should know. For the World Wide Web (WWW), HTML forms a triad of essential web technologies, together with Cascading Style Sheets (CSS) and JavaScript (JS). Without the knowledge of it (or XHTML - a variation of HTML) nobody can create as much as one webpage looking even slightly prettier than a page of unformatted plain text. The WWW, as we know it, would never exist without the invention of HTML. This standard markup language describes the structure of a web page semantically, and originally included cues for the appearance of the document. It consist of tags, and attributes that tell the browser how to render the web page. A web browser receives HTML documents from a web server or locally stored on a device - and then it renders them into several web pages.
